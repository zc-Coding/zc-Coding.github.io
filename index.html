Around the Web

My Site 2021

Twenty One Game

This project is a console application that can play a twenty one game with data loggin of cards dealt to a text file log. Super classes are used so that this app is expandable to add additional games that the same players and dealers with the same bankroll can switch between.

Newsletter Application

The newsletter app uses MVC scaffolding to make a simple sign up page for a newsletter than stores the information in a database. The admin can visit thier own page that allows to edit and delete subscribers

Car Insurance Quote Calculator

This is a console app that prompts the user to put in some basic information about their driving history and current car. The the app takes the input and run it through a string of logic and returns a quote. It also stores all inputs with the quote on an admin page for an employee to contact a potential client.

Ralphs Racing Rentals

Ralphs Rental is a one-page site that uses HTML, CSS, jQuery, and JavaScript. The page fades in upon loading and fades out when navigating somewhere else using jQuery for the effects. The page also contains one embedded video that is responsive to the size of the page and plays asynchronos. The bottom has a form for the customer to input contact info and use a drop down menu to pick which car they are interested in.

CSS Animation Demo

This is a simple demo page that uses CSS animations that trigger upon hovering of a sleeping smiley face image, when hovered over they change into a yellow smiley face and bounces up and down.

Favorite Color Form

This page is visually very simple, it contains a bulleted list of colors with an input form under the list that asks your favorite color. Upon entering a color it iterates through the colors on the page as a lst and check if it mathes anything. if it's a mtach the logic will return "(input color) is a great color!" if not it returns "Please enter a color as shown above".
